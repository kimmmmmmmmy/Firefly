## 一、注册账号

1.  🔗 **访问注册入口**：前往 https://api.aixiaobo.cn/login 进行申请。
2.  📝 **填写信息**：填写自己的相关信息，通过邮箱接收验证码即可完成注册。
3.  🔐 **后续登录**：注册成功后，后续使用用户名或者邮箱登录即可。

## 二、创建自己的令牌密钥

1.  **完成注册**：填写基本信息，完成注册。
2.  **创建令牌**：注册成功之后，点击“创建API令牌”。

## 四、如何充值消费

1.  **进入钱包**：因为是新账号还没有余额，需要进行充值才能使用。我们回到控制台，点击钱包。

<img src="https://img.aixiaobo.cn/image/API%20%E4%BB%A4%E7%89%8C%E5%88%9B%E5%BB%BA%E6%95%99%E7%A8%8B%EF%BC%88%E9%80%9A%E7%94%A8%E7%89%88%EF%BC%89_AI%E5%B0%8F%E6%B3%A2-image-4.png?e=9000000000&token=dLq9-r2FOwOJvQD81bmEpSNQXrE16AAp5zgi0RBW:kAV2uUuyHYixGQt3LbKIize92Z8=" style="height:300px !important;">
